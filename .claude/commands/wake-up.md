# /wake-up

Reads relevant context to prepare for prompts.

## Usage

`/wake-up <optional-prompt>`

## Instructions

In the absence of `<optional-prompt>`, you should

- Read AGENTS.md
- Read ./docs/DESIGN.md

Then you should:

- figure out where we left off in the project
- figure out if we have drifted from the design, or if anything in AGENTS.md seems out-of-date
- prompt the user with your findings and ask the user what we are going to work on

In `<optional-prompt>` is provided, use it to guide you in your context gathering and response.
